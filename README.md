# Advent of Code 2023

## Building and Running
### Option A: Cabal Project
```
cabal run Day1
```
### Option B: GHC
**Compile**
```
ghc Day1/Main.hs
```

**Run**

Linux
```
./Day1/Main
```

Windows
```
.\Day1\Main.exe
```
## Option C: GHCI
**Load into GHCI**
```
ghci Day1/Main.hs
```
**Run in GHCI shell**
```
main
```
**Or**
```
part1
```
**Reload after changes**
```
:r
```
(Then you can rerun `main` with the changes)
